CREAR LA TABLA PARA PINTAR DINAMICAMENTE: tipo form horizontal
  • CREAR ARRAY DE USER FICTICIOS (REALMENTE LEERA DE LA DB)
  • CON SU BOTON EDITAR: Que habilite los input del user
  • Al clicar editar, desaparecera editar(TIPO AL MOSTRAR QUE HIZO BORJA), y aparecera GUARDAR (AL CLICAR, envia los cambios a la DB)

  • LUEGO CADA REGISTRO SE REPETIRA CON NGFOR del ARRAY (*ngFor="let user of usuarios")
CREAR BOTON ACTUALIZAR, CON UNA QUERY QUE VUELVA A LEER LOS USUARIOS DE LA DB PARA ACTUALIZAR
